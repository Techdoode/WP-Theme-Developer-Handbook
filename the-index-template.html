What is the Index Template?

The Index Template, index.php, serves several functions in a WordPress theme. First, it is one of two required theme files. Secondly, it serves as a pattern for several optional files. Lastly, it is the fallback file in case the optional files are missing. While it is possible to get by with just these two files, creating a few additonal files will allow you to take full advantage of the power of WordPress. The Index Template includes these additional files at the appropriate time. Additionally, the Index Template contains The Loop, which is the engine that drives the listing of your Posts.
